# TechNova Solutions - Responsive Company Website (Tailwind CSS)

## Project Description

TechNova Solutions is a modern, responsive multi-page company website developed using HTML5 and Tailwind CSS. This project was created as part of the Week 3 - Day 1 Tailwind CSS Assignment.

The website was converted from a traditional CSS-based design into a Tailwind CSS utility-first design while maintaining a clean, professional, and responsive user interface.

Each page includes a responsive layout, modern design, and consistent navigation.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `home.html` in your web browser.
4. No installation or build process is required because the project uses the Tailwind CSS CDN.
