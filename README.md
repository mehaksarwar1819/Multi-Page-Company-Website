# TechNova Solutions - Responsive Company Website

## Project Description

TechNova Solutions is a responsive multi-page company website developed as the Final CSS Assignment. The project demonstrates the use of modern HTML5 and CSS3 techniques to create a professional, responsive, and user-friendly company website.

The website consists of four pages:
- Home
- About
- Services
- Contact

The project focuses on responsive design, modern layouts, animations, reusable components, and clean code organization.

---

## Features

- Responsive Navigation Bar
- Sticky Header
- Modern Hero Section
- Responsive Layout using Flexbox & CSS Grid
- Professional Color Scheme
- Google Fonts
- Services Cards
- Team Members Section
- Testimonials
- Pricing Plans
- Contact Form
- Professional Footer
- Smooth Hover Effects
